# BeFit - Blazor

## Overview

Application for tracking trainings.

## Getting Started

1. Clone the repository.

2. Build:
```bash
dotnet build
```

3. Apply migrations (if using EF Core):
```bash
dotnet ef database update
```

4. Run locally:
```bash
dotnet watch run
``` 

5. Open the app at http://localhost:5276/
